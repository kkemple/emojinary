Pictionary with emoji right in

Slash Commands

/emojinary

Calling emojinary by itself will call /emojinary current.

/emojinary new

Use the new command to create a new emojinary for your team to solve.

/emojinary new emojinary=[] hint=[] answer=[]

Options
  • emojinary (required): the emoji string that makes up your emojinary emojinary=[:flag-it: :horse: :punch: :flag-ru:]
  • answer (required): the answer to your emojinary (helpful hint: always spell out numbers in answers!) answer=[rocky four]
  • hint (optional): a helpful hint for your team hint=[i must break you!]

/emojinary current

Use the current command to see the current emojinary in play. If there isn't one in play, it will return the result of the last solved emojinary if there is one.

/emojinary hint

Use the hint command to see if the current emojinary has a clue if you're having a tough time figuring one out.

/emojinary solve

Use the solve command once you're ready to take a crack at an emojinary.

/emojinary solve rocky four
