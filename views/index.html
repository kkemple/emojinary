Slash Commands

/emojinary

Calling emojinary by itself will call /emojinary current.

/emojinary new

Use the new command to create a new emojinary for your team to solve.

/emojinary new emojinary=[] hint=[] answer=[]

Options
  • emojinary (required): the emoji string that makes up your emojinary
    emojinary=[:flag-it: :horse: :punch: :flag-ru:]
  • answer (required): the answer to your emojinary (helpful hint: spell out numbers!)
    answer=[rocky four]
  • hint (optional): a helpful hint for your team
    hint=[i must break you!]

/emojinary current

Use the current command to see the current emojinary in play.
If there isn't one in play, it will return the result of the last solved emojinary if there is one.

/emojinary hint

If you're having a tough time, use the hint command to see if there is a clue.

/emojinary hint => i must break you!

/emojinary solve

Use the solve command once you're ready to take a crack at an emojinary.

/emojinary solve rocky four

/emojinary random

Use the random command to pick an emojinary from out database.

/emojinary random

/emojinary forfeit

Use the forfeit command when you just can't figure it out.

/emojinary forfeit

/emojinary stats

Use the stats command to see how you and your teammates are doing!

  • Attempts, Solves, Forfeits, Creates

/emojinary stats

/emojinary list

Use the list command to view all the emojinary you and your team have solved.

/emojinary list
